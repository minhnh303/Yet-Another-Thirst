**Yet Another Thirst** is a simple mod that adds a thirst mechanic to the game, but that fits well in modpacks (especially **aesthetics**-oriented ones) by adding diverse features, compatibility with a lot of often-used mods, and more.

This mod is heavily inspired by the other thirst/survival mods out there, including Dehydration, Tough as Nails, and Cold Sweat.

*Note: This project is rebuilt based on the [ThirstWasTaken](https://github.com/mlus-asuka/Thirst-Remade/) mod.*

To learn more about this mod, you can visit the [Wiki](https://github.com/minhnh303/Yet-Another-Thirst/wiki).
